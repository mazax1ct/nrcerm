Вопрос-ответ

Записаться на консультацию специалиста и исследования можно по телефону контакт-центра (812) 339-39-39 (с 9:00 до 18:00 по будням) или заполнив форму для записи на сайте.

Записаться на приём по полису ОМС можно при наличии направления из поликлиники по месту жительства пациента. Запись осуществляется по телефону (812) 339-39-39 с 9:00 до 18:00 по будням.

Чтобы записаться на приём по полису ДМС, необходимо обратиться в вашу страховую компанию.

Медицинская помощь сотрудникам МЧС России и членам их семей оказывается по месту жительства в соответствии с программой государственных гарантий бесплатного оказания гражданам медицинской помощи (ОМС). При отсутствии возможности (документально подтверждённой в медицинских документах) оказания медицинской помощи по месту службы, пациент может получить медицинскую помощь во ВЦЭРМ им. А.М. Никифорова МЧС России. Для этого необходимо обратиться в медицинское подразделение территориального органа МЧС России и получить направление во ВЦЭРМ им. А. М. Никифорова МЧС России. Имея направление, записаться на приём можно по телефону (812) 339-39-39 с 9:00 до 18:00 по будням.

По всем возникающим техническим вопросам Вы можете обратиться на нашу горячую линию по бесплатному номеру телефону: - +7 (800) 550-72-65 или на электронную почту: service@digital-razor.ru.

Менеджеры нашей компании постараются помочь решить вопрос максимально оперативно. Если задача требует привлечения технических специалистов, мы проведем первичную диагностику и сообщим о дальнейших шагах удобным для Вас способом.
